SWILR Motivation
================
A bar means "is affected" or "is compared" or "is used".

An arrow means "is updated" or "receives".


add      : WIA
clean    : W
clone    : RLWA, RLA
checkout : LWA
commit   : ILA, WILA, WLA
init     : L
fetch    : RLA
pull     : RLWA
push     : LRA
rm       : WIA, I
revert   : WILA
